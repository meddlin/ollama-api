# ollama-api
An API wrapper for Ollama


## Getting Started

1. Install Ollama

[https://ollama.com/](https://ollama.com/)

2. Pull a model

```bash
ollama run llama3.1
```

3. Make an environment and have fun

### Windows

```bash
# make an environment
python -m venv .

# activate environment
./bin/Activate.ps1
```

### Linux & MacOS

```bash
# make an environment
python -m venv .

# activate environment
source ./bin/activate
```
